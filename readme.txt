
*** 기본 사용법 ***

* 서로이웃 관리 자동화 프로그램은 "서로이웃 신청"과 "이웃정리" 두가지 메뉴를 제공합니다.

* 서로이웃은 다양한 경로를 통해 얻은 소스를 바탕으로 서이추 신청할 수 있습니다.
  (포스트에 공감, 이웃의 커넥트, 카페 게시글 작성자[베타서비스], 파일 직접 등록)

1. 아이디와 비밀번호를 입력하세요.

2. 주소창에 서이추 소스별 URL 주소를 정확하게 입력하세요.

3. 파일 직접 업로드는 엑셀 파일과 텍스트 파일 2종류를 지원합니다.

4. 서로이웃 신청 메시지는 단순한 문구보다는 좀 더 정성들여 작성하시길 권장합니다.

5. '시작' 버튼을 클릭하면 프로그램이 실행됩니다.

6. 프로그램을 중단하려면 '중지' 버튼을 클릭하세요.

7. 사용기한 내에 추가되는 기능이나, 수정되는 부분은 무료로 자동 업데이트 됩니다.

8. 좀 더 자세한 사항은 "도움말" 메뉴에서 확인하세요!
